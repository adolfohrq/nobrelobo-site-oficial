## Contribuindo

Ao contribuir com este projeto, certifique-se de:
1. Seguir todos os padrões descritos nesta documentação
2. Ler o [Coding Standards](./12-coding-standards.md) antes de submeter código
3. Adicionar testes quando aplicável (ver [Testes](./10-testing.md))
4. Documentar componentes novos seguindo os padrões existentes
